## 📌 Project Overview

This project analyzes the Python programming language as an open-source software system. It covers its origin, license, ethical considerations, Linux footprint, ecosystem, and comparison with proprietary software.

The project also includes five shell scripts demonstrating Linux command-line skills and automation.

## 💻 Shell Scripts

### 🔹 Script 1 — System Identity Report

Displays system information such as kernel version, user, uptime, and OS details.

### 🔹 Script 2 — FOSS Package Inspector

Checks if Python is installed and displays its version and details.

### 🔹 Script 3 — Disk and Permission Auditor

Analyzes system directories and displays permissions and disk usage.

### 🔹 Script 4 — Log File Analyzer

Reads log files and counts occurrences of keywords like "error".

### 🔹 Script 5 — Open Source Manifesto Generator

Generates a personalized manifesto based on user input.

## ⚙️ How to Run Scripts

### Step 1: Open terminal

### Step 2: Give permission

```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

### Step 3: Run scripts

```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
```
